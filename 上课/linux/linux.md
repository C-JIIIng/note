分区命名方案

+ /dev/xxyN（/dev/sdb1）

+ /dev  所有设备文件的目录名
+ xx 表示分区所在的设备的类型，如：sd（硬盘）
+ y 表示分区所在的设备 如 /dev/sdb（第2个硬盘）
+ N 表示分区
  + 1-4（主分区或扩展分区）
  + 5- （逻辑驱动器）
